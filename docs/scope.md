# Scope — Public Education and Showcase Only

## What this repository is

`ai-in-action-roadmap` is the **public** planning and architecture documentation for the **AI in Action** series: production-ready personal case studies meant for learning and professional credibility.

Audience: engineers, architects, hiring managers, and peers.

Purpose: demonstrate engineering judgment, architecture, AI-assisted delivery, CI/CD, and cloud deployment through live MIT-licensed demos (RetireCheck, SleepCheck, and future Check apps).

## What this repository is not

This repository does **not** include private commercial strategy.

Do not document here:

- Pricing, packaging, or paid tiers  
- Payment providers or billing flows  
- Willingness-to-pay analysis  
- Open-core sales pitches or commercial product roadmaps  
- Private business plans  

Those topics, if discussed at all, stay private between the owner and trusted agents — outside this public tree.

## How to treat optional future themes

Optional accounts, cloud sync, shared UI patterns, deeper local history, accessibility, testing, and observability are welcome **when framed as engineering learning**, not as paywalls or commercial packaging.
